
.. _operations:

List of operations
==================

The term *Operation* is used as an abstraction to indicate a process taking one 
or many inputs and returning one or many outputs. 
This concept is the basis of the :ref:`experiment <experiment>`, which consists 
of a series of operations.

For example, the operation :ref:`houghtransform` takes two inputs: 
a pattern and a value for the resolution. 
After performing the transformation, it returns one output, 
the resultant Hough transform.

The following sections gives a list of the available operations. 
There are classified by their group and category. 
For a description of the categories refer to the :ref:`categories` of the 
experiment engine.
